Getting started

1. Install dependencies:

```powershell
npm install
```

2. Build once:

```powershell
npm run build
```

3. Dev server (live reload):

```powershell
npm run start
```

To publish to GitHub Pages, create a repository and push this code; the included workflow will build and deploy the `_site` folder.
